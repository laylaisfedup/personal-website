## Customization

### Update Content
Edit the HTML files to add your personal information, projects, books, and thoughts.

### Change Colors
Edit `style.css` to modify the color scheme. Current palette:
- Background: `#ffffff` (white)
- Text: `#333333` (dark grey)
- Links: `#6b7280` (pale grey)
- Links hover: `#1a1a1a` (dark)

### Add Blog Posts
1. Create `blog/` folder
2. Add HTML files for each post
3. Link them in `blog.html`

## Local Development

Simply open `index.html` in your web browser to preview the site locally.
